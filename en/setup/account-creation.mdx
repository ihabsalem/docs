## Troubleshooting Common Issues üîß

### Email Verification Problems:
**Issue**: "Email not received"
**Solutions**:
- Check spam/junk folders
- Add chatwiser.co to your email whitelist
- Try a different email address
- Contact support for manual verification

### Login Difficulties:
**Issue**: "Can't log in to account"
**Solutions**:
- Verify email address is correct
- Reset password using "Forgot Password" link
- Clear browser cache and cookies
- Try incognito/private browsing mode

### Profile Setup Errors:
**Issue**: "Can't save business information"
**Solutions**:
- Check all required fields are filled
- Ensure phone number format is correct
- Try refreshing the page
- Use different browser if issues persist

### Dashboard Not Loading:
**Issue**: "Dashboard appears blank"
**Solutions**:
- Check internet connection
- Disable browser extensions temporarily
- Clear browser cache
- Try different browser
