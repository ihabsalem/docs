    ### What's the difference between a bot and live chat?
    
    **Bot (Automated):**
    - Responds instantly 24/7
    - Handles routine questions
    - Follows pre-defined flows
    - Can collect information and process orders
    - Never gets tired or takes breaks
    
    **Live Chat (Human Agent):**
    - Handles complex questions
    - Provides personalized service
    - Can make decisions and exceptions
    - Available during business hours
    - More expensive but more flexible
    
    **Best Practice**: Use bots for routine tasks and humans for complex issues.

    4. Connect buttons to actions
    5. Test all paths

## Technical Support Questions üÜò

### Common Issues

**Q: My bot isn't responding to messages. What's wrong?**

**Troubleshooting Steps:**
1. **Check Bot Status**: Is it enabled/active?
2. **Verify Page Connection**: Is Facebook page still connected?
3. **Test Triggers**: Are keyword triggers working?
4. **Review Flow Logic**: Any broken connections?
5. **Check Permissions**: Does chatwiser have necessary permissions?

**Quick Fix**: Try disconnecting and reconnecting your Facebook page.

**Q: Customers say they can't see my bot messages. Why?**

**Common Causes:**
- **Message Type**: Some content blocked by Facebook
- **Page Settings**: Messages disabled in Facebook settings
- **User Blocks**: Customer blocked your page
- **Technical Issues**: Temporary Facebook API issues

**Solutions:**
1. Check Facebook page message settings
2. Test with different Facebook account
3. Review message content for policy violations
4. Contact support if issues persist

**Q: How do I backup my bot flows and data?**

**Automatic Backups:**
- **Daily Backups**: All data backed up automatically
- **Version History**: Previous versions of flows saved
- **Export Options**: Download flows and data anytime
- **Cloud Storage**: Redundant storage across regions

**Manual Export:**
1. Go to **Settings** ‚Üí **Data Export**
2. Select what to export (flows, contacts, conversations)
3. Choose format (JSON, CSV, PDF)
4. Download when ready
